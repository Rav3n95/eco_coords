# ECO COORDS
- Save coords server side in txt file
- Copy to clipboard

## COMMANDS:
- '/eco_coord' ON / OFF
- '/resetcoords' set default settings 

## CONTROLS

- SCROLL or NUMPAD + / -: change marker z position
- SHIFT + SCROLL: change marker size
- SHIFT + CTRL + SCROLL: change marker type
